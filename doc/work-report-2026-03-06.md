### 3. プロフィールサイト構築

#### 情報収集元
- lit.link/morune（ブランド情報・リンク一覧）
- note.com/mizuhana_aoi（プロフィール・記事情報・画像）
- techbookfest.org（書籍情報・表紙画像）

#### 最終サイト構成（エディトリアル・雑誌風）
| セクション | 内容 |
|-----------|------|
| Hero | 左: キャッチコピー + CTA / 右: 書籍カバー画像（ブランドイメージ） |
| 01 Works | 書籍・note記事（画像カード） + ストア・SoundCloud・Instagram（アイコンカード） |
| 02 Profile | 左: レコード画像（Unsplash） / 右: 小さい顔写真 + 経歴・専門タグ |
| 03 Links | ダーク背景にグリッド罫線で6件のリンクを配置 |
| 04 Contact | CTA（Instagram DM / X で問い合わせ） |

#### 最終デザイン仕様
- **スタイル**: エディトリアル・雑誌風（非対称レイアウト、大胆なタイポグラフィ、余白重視）
- **フォント**: Playfair Display（見出し・セリフ体）/ Noto Sans JP（本文）
- **配色**: オフホワイト背景（#faf8f5）+ テラコッタアクセント（#c47a5a）+ ダーク（#1a1a1a）
- **ナンバリング**: 01〜04 で雑誌の目次風にセクション番号を付与
- **アニメーション**: スクロール連動フェードイン（IntersectionObserver）+ パララックス（画像がスクロールでずれて動く）
- **ナビ**: mix-blend-mode: difference による透過トップバー
- **レスポンシブ**: モバイル対応（ハンバーガーメニュー、1カラムフォールバック）
- **OGPメタタグ**: 設定済み
